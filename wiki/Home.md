# Kokoro TTS MCP Server Wiki

Welcome to the **Kokoro TTS MCP Server** documentation wiki!

## Overview

The Kokoro TTS MCP Server is a Model Context Protocol (MCP) server that provides text-to-speech capabilities using the Kokoro TTS model. It features a female voice by default (`af_heart`) and runs **100% locally** with no Python dependencies required.
